3D Chessboard Design – Autodesk 3ds Max

This 3D modelling project showcases a photorealistic chessboard and full chess set created entirely in Autodesk 3ds Max. Starting from a simple Plane blueprint, I used Line and Box primitives to construct the board layout. Each tile was refined with Bevel modifiers to achieve clean, professional edges. The chess pieces pawns, bishops, knights, rooks, queens, and kings were modeled using Lathe modifiers, allowing for smooth and symmetrical surfaces. To enhance realism, I strategically positioned Omni lights to create dynamic shadows and highlights that bring depth to the scene. The result is a polished, animation-ready chessboard designed for use in games, visualizations, or interactive 3D environments. This project reflects my skills in 3D modeling, lighting, and detail-oriented design.
